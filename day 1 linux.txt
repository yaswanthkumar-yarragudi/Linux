DAY 1: LINUX INTRODUCTION & DISTRIBUTIONS

What is Linux?
Linux is an open-source operating system kernel created by Linus Torvalds.

Free and open-source
Secure and stable
Widely used in servers and DevOps

Why Learn Linux?

Most servers run on Linux
Important for DevOps and Cloud roles
Strong command-line control

Linux Architecture

User → Shell → Kernel → Hardware

What is a Linux Distribution?

A Linux distribution is a complete OS built using:

Linux kernel
System tools
Package manager

Popular Linux Distributions

Ubuntu
Beginner-friendly
Uses apt package manager
Debian
Very stable
Good for servers
Rocky Linux / CentOS
Enterprise-level
Uses yum or dnf
Kali Linux
Used for security testing
Arch Linux
Advanced users
Uses pacman

Package Managers

Ubuntu/Debian → apt
Rocky/CentOS → yum / dnf
Arch Linux → pacman

Key Takeaways

Linux is widely used in DevOps
Different distributions serve different purposes
Ubuntu is best for beginners
