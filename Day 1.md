# 📅 Day 1: Linux Introduction & Distributions

## 🐧 What is Linux?
Linux is an open-source operating system kernel created by Linus Torvalds.

<p align="center">
  <img src="https://raw.githubusercontent.com/devicons/devicon/master/icons/linux/linux-original.svg" width="120"/>
</p>

- Free and open-source  
- Secure and stable  
- Widely used in servers and DevOps  

---

## 💡 Why Learn Linux?
- Most servers run on Linux  
- Essential for DevOps & Cloud roles  
- Powerful command-line interface (CLI)  

---

## 🧱 Linux Architecture

User → Shell → Kernel → Hardware

<p align="center">
  <img src="https://www.interviewbit.com/blog/wp-content/uploads/2022/06/Linux-Architecture-1024x728.png" width="300"/>
</p>

- User: Interacts with the system  
- Shell: Command interpreter (e.g., Bash)  
- Kernel: Core of OS  
- Hardware: Physical components  

---

## 🧾 What is a Linux Distribution?
A Linux distribution (distro) is a complete OS built using:
- Linux kernel  
- System tools  
- Package manager  

---

## 🌍 Popular Linux Distributions

### 1. Ubuntu
<img src="https://assets.ubuntu.com/v1/29985a98-ubuntu-logo32.png" width="120"/>

- Beginner-friendly  
- Uses `apt`  

### 2. Debian
<img src="https://raw.githubusercontent.com/devicons/devicon/master/icons/debian/debian-original.svg" width="120"/>

- Stable and reliable  
- Preferred for servers  

### 3. Rocky Linux / CentOS
<img src="https://raw.githubusercontent.com/devicons/devicon/master/icons/redhat/redhat-original.svg" width="120"/>

- Enterprise-level  
- Uses `yum` / `dnf`  

### 4. Kali Linux
<img src="https://raw.githubusercontent.com/devicons/devicon/master/icons/linux/linux-original.svg" width="120"/>

- Used for security testing  

### 5. Arch Linux
<img src="https://raw.githubusercontent.com/devicons/devicon/master/icons/archlinux/archlinux-original.svg" width="120"/>

- Advanced users  
- Uses `pacman`  

---

## ⚙️ Package Managers

| Distribution     | Package Manager |
|------------------|----------------|
| Ubuntu/Debian    | `apt`          |
| Rocky/CentOS     | `yum` / `dnf`  |
| Arch Linux       | `pacman`       |

---

## 🧠 Key Takeaways
- Linux is the backbone of DevOps  
- Choose distro based on use case  
- Start with Ubuntu
